# 问题
假设你正在爬楼梯。需要 n 步你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。
# 程序
```C
int climbStairs(int n) {
    int i;
    int a = 1,b = 2;
    int count;
    if(n == 0 || n == 1 || n == 2)
        return n;
    for(i = 3;i <= n;i++)
    {
        count = a + b;
        a = b;
        b = count;
    }
    return count;
}
```
# 心得
因为每次可以爬1或2个台阶，所以在0，1，2个台阶的时候，直接等于台阶数；3个及3个以上的时候，每次1或2个台阶，所以令a为1，b为2，计数一开始就是a+b，然后互相赋值，在原来的基础上再计数。
